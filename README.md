# bale-news-bot
# 🤖 ربات خبرخوان مهر

ربات خبرخوان برای ارسال روزانه اخبار از خبرگزاری مهر در کانال بله.

## ⚙️ تنظیم در GitHub

### ۱. افزودن Secrets

در Settings > Secrets and variables > Actions:

| Name | Value |
|------|-------|
| `BOT_TOKEN` | توکن ربات از @BotFather |
| `CHANNEL_ID` | آیدی عددی کانال |
| `AD_LINK` | لینک تبلیغ (@GoldChi) |

### ۲. فعال‌سازی Actions

به تب Actions بروید و workflow را فعال کنید.

### ۳. اجرای دستی

از تب Actions می‌توانید اجرای دستی هم داشته باشید.

## ⏰ زمان اجرا

هر روز ساعت 5:30 صبح UTC (9:00 صبح به وقت ایران)

## 📝 دستورات

| دستور | توضیح |
|-------|-------|
| `/start` | شروع |
| `/help` | راهنما |
| `/status` | وضعیت |
| `/stats` | آمار |
| `/refresh` | ارسال دستی اخبار |
| `/test` | تست |
| `/setad` | تنظیم تبلیغ |

## 🔒 امنیت

- توکن‌ها در GitHub Secrets ذخیره می‌شوند
- فایل `.env` در گیت آپلود نمی‌شود
- دیتابیس‌ها بعد از هر اجرا آپدیت می‌شوند
